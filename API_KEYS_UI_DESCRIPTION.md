## Features Demonstrated

### Table Features
- **Name**: Descriptive name given by user
- **Key Preview**: First 4 and last 4 characters of the key (e.g., "dGhp...bW8")
- **Status Chips**:
  - ğŸŸ¢ Green "Active" for valid, unexpired keys
  - ğŸ”´ Red "Expired" for keys past their expiration date
  - âšª Gray "Inactive" for manually deactivated keys
- **Expires Column**: Shows either:
  - "Never" for infinite lifetime keys
  - Date (e.g., "2026-02-01") for keys with expiration
  - "Expired" for already-expired keys
- **Last Used**: Timestamp of last API call or "Never"
- **Actions**: Delete button (trash icon)

### Create Dialog Features
- **Lifetime Options**:
  - Never expires (infinite)
  - Quick options: 7, 30, 90, 365 days
  - Custom: any number of days
- **Validation**:
  - Name is required
  - Custom days must be positive number
- **Security**:
  - Key shown only once
  - Warning message about copying
  - Copy to clipboard button (ğŸ“‹)

### User Experience
- Table is empty state shows: "No API keys yet. Create your first API key to get started."
- Loading spinner while fetching keys
- Error alerts for failed operations
- Confirmation dialog before deletion
- Success feedback after operations
